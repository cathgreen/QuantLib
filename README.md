# QuantLib
A toy Quantitative pricing library written in C++.
The last version is QuantLib_0.9.0

============================
To build a dynamic lib: 
  in ./QuantLib_0.9.0/, run
  $ cmake .. 
  then
  $ make
 
============================
To test the lib:
  edit code in testQL.cpp
  in ./testsQL_res/, run
  $ cmake .. 
  then
  $ make
